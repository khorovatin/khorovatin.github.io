# khorovatin.github.io

Published output for [ken.horovatin.net](https://ken.horovatin.net/).

## Purpose

This repository contains the generated static site that is served by GitHub Pages for the custom domain `ken.horovatin.net`.

## Source repository

The source files for the site live in:

- [`khorovatin/personal-website`](https://github.com/khorovatin/personal-website)

That repository contains:

- the Hugo/Blogdown source
- content and configuration
- theme files
- deployment workflows
- documentation for editing and publishing

## Important

Do **not** make normal content or theme edits in this repository.

Changes should be made in the source repository and deployed automatically to this repository through GitHub Actions.

## Deployment

This repository is updated automatically by the deployment workflow in the source repository.
